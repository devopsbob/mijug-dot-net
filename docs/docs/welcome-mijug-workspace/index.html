Workspace Home

Welcome to MIJUG .NET Workspace

This is a TypeScript and Jekyll development workspace designed for modern web development with WSL integration.

Features

  • TypeScript Development: Full TypeScript support with IntelliSense
  • Jekyll Documentation: Static site generation for project documentation
  • WSL Integration: Optimized for Windows Subsystem for Linux
  • Code Quality: ESLint and Prettier integration
  • CI/CD Ready: GitHub Actions workflows included

Quick Start

  1. Install dependencies:

    npm install
cd docs && bundle install

  2. Start development:

    # TypeScript development
npm run dev

# Jekyll development
npm run dev:jekyll

Project Structure

  • src/ - TypeScript source code
  • docs/ - Jekyll documentation site
  • tests/ - Test files
  • scripts/ - Build and deployment scripts
  • dist/ - Compiled TypeScript output
