Getting Started

Welcome to the MIJUG .NET workspace - a comprehensive development environment that combines modern TypeScript development with powerful Jekyll documentation capabilities, all optimized for WSL (Windows Subsystem for Linux) integration.

📚 Looking for detailed commands? All specific installation steps, commands, and workflows are documented in the main private repository README file. We’d love to help you get started with full access - reach out via the contact button below and let’s schedule a friendly conversation about your development goals and how we can support your success!

🔍 Want in-depth tutorials? Check out our detailed Getting Started with TypeScript and Jekyll guide for comprehensive explanations and code examples.

🚀 Workspace Features Overview

Development Environment

  • TypeScript Development: Full-featured TypeScript workspace with IntelliSense, debugging, and hot reload capabilities
  • Jekyll Documentation Platform: Static site generation with advanced markdown features, live preview, and draft management
  • WSL Integration: Seamlessly optimized for Windows Subsystem for Linux (Ubuntu) development
  • VS Code Optimization: Pre-configured workspace with essential extensions and settings

Testing & Quality Assurance

  • Multi-Level Testing: Jest unit tests, Playwright E2E testing, and WCAG 2.1 AA accessibility compliance
  • Cross-Browser Compatibility: Automated testing across Chromium, Firefox, and WebKit browsers
  • Code Quality Tools: ESLint linting, Prettier formatting, and automated code quality checks
  • Performance Monitoring: Built-in performance optimization and cache validation tools

Content Management

  • FrontMatter CMS: Visual content management system integrated directly into VS Code
  • Jekyll Drafts System: Complete draft workflow for content creation and review
  • Media Management: Integrated asset management with automatic optimization
  • Taxonomy Organization: Advanced tagging and categorization systems

Performance & Optimization

  • Parallel Build System: Optimized build processes with TypeScript and Jekyll running simultaneously
  • Intelligent Caching: Advanced caching strategies for faster development cycles
  • Memory Optimization: Configurable memory allocation for large-scale projects
  • CI/CD Ready: GitHub Actions workflows with performance-optimized pipelines

🛠️ Core Technologies & Standards

Frontend Technologies

  • TypeScript: Strongly-typed JavaScript with ES2022 target support
  • Jekyll: Ruby-based static site generator with Liquid templating
  • Playwright: Cross-browser automation and testing framework
  • Jest: JavaScript testing framework with snapshot testing

Development Tools

  • Visual Studio Code: Primary IDE with extensive extension support
  • ESLint: JavaScript/TypeScript linting with configurable rules
  • Prettier: Opinionated code formatting for consistent style
  • Ruby LSP: Ruby language server for enhanced Jekyll development

Quality & Accessibility

  • axe-core: Accessibility testing engine for WCAG compliance
  • WCAG 2.1 Guidelines: Web Content Accessibility Guidelines Level AA
  • Lighthouse: Performance and quality auditing
